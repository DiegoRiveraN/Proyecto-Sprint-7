# Proyecto Sprint 7 – Análisis exploratorio de datos

Aplicación web desarrollada con Streamlit que permite explorar un conjunto de datos
de anuncios de venta de vehículos en EE. UU.

## Funcionalidades
- Vista previa del dataset
- Información estadística general
- Histogramas interactivos
- Gráfico de dispersión precio vs kilometraje

## Tecnologías utilizadas
- Python
- Pandas
- Plotly Express
- Streamlit
